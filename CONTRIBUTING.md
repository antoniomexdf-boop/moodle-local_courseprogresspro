# Contributing

## English

Thanks for contributing to `local_courseprogresspro`.

### Workflow

1. Fork the repository.
2. Create a branch from `main`.
3. Implement your change with clear commit messages.
4. Validate in Moodle 4.1+.
5. Open a Pull Request with summary, test steps and screenshots if UI changed.

## Español

Gracias por contribuir a `local_courseprogresspro`.

### Flujo de trabajo

1. Haz fork del repositorio.

3. Implementa el cambio con commits claros.
